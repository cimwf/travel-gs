🔒 密码隐藏状态

闭眼 + 斜线 = 密码不可见

• 图标颜色：#8E8EA0（灰色）

• 点击区域：32×32px

• Hover 背景变浅蓝

🔓 密码可见状态

睁眼 + 瞳孔 = 密码可见

• 图标颜色：#8E8EA0（与闭眼一致）

• 无斜线 = 可以查看

• 圆形瞳孔增加辨识度

💡 设计要点

统一配色

两种状态颜色一致 (#8E8EA0)

形态区分

闭眼有斜线，睁眼有瞳孔

视觉简洁

线条风格，与输入框协调

交互反馈

Hover 背景变浅灰
