📐 设计说明

登录流程

1. 进入小程序
→
2. 手机号授权页
→
3. 微信获取手机号
→
4. 完善信息页
→
5. 进入首页

📱 页面1: 手机号授权页

  • 品牌Logo和slogan展示
  • 价值引导文案（发布行程、收藏景点等）
  • 微信一键登录按钮（getPhoneNumber）
  • 协议链接（用户协议、隐私政策）
  • 渐变背景，与首页风格统一

👤 页面2: 完善信息页

  • 头像必填（使用微信头像/自定义上传）
  • 昵称必填（微信nickname能力）
  • 性别选填（默认保密）
  • 返回按钮（可取消）
  • 完成注册按钮提交

🔧 技术实现

  • 使用微信 getPhoneNumber 获取手机号
  • 使用微信 chooseAvatar 获取头像
  • 云函数验证手机号并创建用户
  • 头像上传到云存储
  • 用户信息存入云数据库

✨ 设计亮点

  • 渐变背景统一风格
  • 卡片式表单设计
  • 头像预览+点击更换
  • 性别选项直观易选
  • 必填项明确标识
