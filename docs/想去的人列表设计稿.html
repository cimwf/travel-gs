🎨 设计说明

想去的人列表页

样式统一

  • 卡片样式与地点详情页一致
  • 相同的信息结构
  • 相同的按钮样式
  • 相同的交互效果

页面功能

  • 展示该地点所有想去的人
  • 支持筛选：出发时间/有车/人数
  • 点击卡片进入行程详情页
  • 底部固定"发布我的行程"按钮

交互设计

  • 点击卡片 → 行程详情页（评论版）
  • 点击"申请加入" → 申请表单
  • 点击"邀请他" → 邀请确认
  • 筛选标签切换激活状态

页面跳转链路

  • 地点详情页 → 点击"更多"
  • 想去的人列表 → 点击卡片
  • 行程详情页（评论版）
