📋 产品需求文档

📄 版本：v1.6.0 📅 日期：2026-04-06 👤 作者：产品团队

1. 产品概述

北京去哪玩 是一款面向北京及周边地区的户外旅行社交小程序，帮助用户发现周边好玩的地方，找到志同道合的出行伙伴。

📍
发现地点
收录北京周边户外地点
👥
找搭子
找到同行的伙伴
🚗
拼车出行
有车族提供座位
💬
即时沟通
行程群聊更便捷

2. 登录逻辑

🔐

核心规则

底部导航栏三个Tab（首页、消息、我的）可以直接访问，无需登录。其他所有页面进入前都要判断是否登录，未登录则跳转到登录页。

2.1 页面访问权限

页面 是否需要登录 说明
首页（Tab） 否 可直接访问，展示推荐内容
消息（Tab） 否 可直接访问，未登录显示引导登录
我的（Tab） 否 可直接访问，未登录显示引导登录
登录授权页 否 未登录用户访问
个人信息编辑 是 登录成功后跳转
地点列表 是 需要登录才能浏览
地点详情 是 需要登录才能查看
发布行程 是 需要登录才能发布
行程详情 是 需要登录才能查看/申请
我的行程 是 需要登录查看自己的行程
聊天详情 是 需要登录才能聊天
关注粉丝 是 需要登录查看
评论列表 是 需要登录才能评论
设置 是 需要登录才能设置

用户访问需登录页面

↓
2

↓
3

💬 从消息Tab出发的跳转链路

消息Tab → 聊天详情 （需登录）

👤 从我的Tab出发的跳转链路

我的Tab → 编辑资料
我的Tab → 我的行程
我的Tab → 关注粉丝
我的Tab → 设置

需登录页面 → 登录授权页 → 个人信息编辑页 → 首页Tab
