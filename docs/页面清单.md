## 五、核心用户流程分析

### 流程1：浏览 → 发布行程

```
首页 → 地点详情页 → [登录] → 发布行程页 → 发布成功页 → 行程详情页
```

**状态**：✅ 完整

---

### 流程2：浏览 → 申请加入行程

```
首页 → 地点详情页 → 点击"想去的人"中的行程 → 行程详情页 → [登录] → 申请加入
```

**状态**：✅ 完整

---

### 流程3：处理申请/邀请

```
消息 Tab → 行程通知/邀请消息 → 同意/拒绝
```

**状态**：✅ 完整

---

### 流程4：查看个人内容

```
我的 Tab → 我的行程 → 管理/查看行程
```

**状态**：✅ 完整
